# lcScamp

The langchain based client for my ScampAI project. I am using Langchain 1.0 to manage working with my Ollama based LLMs and working with MCP tools.

See [Langchain](https://reference.langchain.com/python/langchain/langchain/)


## Installation

- uv init
- uv venv
- source .venv/bin/activate


- uv add langchain
- uv add langchain-ollama
- uv add langchain-mcp-adapters
